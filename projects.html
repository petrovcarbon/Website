REST API

Nodejs • MongoDB • TypeScript

Open - github link

A simple REST API that works with mongoDB.

What this project contains

  • Security: The api contains an authentication & session token to check if user is authorized to edit/delete data.
  • GET /users: This route displays the list of users.
  • POST /users: This route accepts JSON data from the client to create a new user.
  • PUT /users/:id: This route updates the information of a user based on the user ID provided by the user in the URL.
  • DELETE /users/:id: This route deletes a user with the specified ID provided by the user.
