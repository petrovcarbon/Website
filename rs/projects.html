REST API

Nodejs • MongoDB • TypeScript

Otvori - github link

Jednostavan REST API koji radi sa mongoDB-om.

Šta ovaj projekat sadrži

  • Bezbednost: API sadrži token za autentifikaciju i sesiju kako bi se proverilo da li je korisnik ovlašćen za uređivanje/brisanje podataka.
  • GET /users: Ova ruta prikazuje listu korisnika.
  • POST /users: Ova ruta prihvata JSON podatke od klijenta za kreiranje novog korisnika.
  • PUT /users/:id: Ova ruta ažurira informacije o korisniku na osnovu korisničkog ID-a koji je korisnik naveo u URL-u.
  • DELETE /users/:id: Ova ruta briše korisnika sa navedenim ID-om koji je korisnik naveo.
